# Tip Calculator

A simple tip calculator app that helps you calculate tips and split bills at restaurants.

## Features

- Enter bill amount
- Select tip percentage (10%, 15%, 18%, 20%, 25%, 30%, or custom)
- Calculate tip amount
- Display total amount to pay
- Clean and user-friendly interface

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How to Run

1. Clone the repository
```bash
git clone https://github.com/Oyekanmi123/Tip-Calculator.git
```
2. Open the project folder
3. Open `index.html` in your browser
